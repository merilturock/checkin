Checkin
===========

![Elgg 3.0](https://img.shields.io/badge/Elgg-3.0-green.svg?style=flat-square)

## Features

- Adds UI for check-ins

## API Key

Make sure your Google API key supports:

 - Maps JavaScript API 
 - Places API
 - Geocoding API

## Note

This directory must be named 'checkin' in your Elgg mod folder.
